WalkRoute is a free, privacy-first pedestrian router. Unlike Google Maps, it uses no API keys, no billing, and no tracking.

  • Pedestrian-optimized paths — routes follow sidewalks, footpaths, park trails, pedestrian zones, and crosswalks via OSRM's foot profile
  • Walking metrics — estimated steps and calories, not just distance and time
  • Multi-stop planning — add up to 5 intermediate stops for errands or walks
  • Alternative routes — compare up to 3 different walking paths
  • 100% client-side — no server, no accounts, runs from static files
  • Global coverage — works anywhere OpenStreetMap has data (worldwide)
