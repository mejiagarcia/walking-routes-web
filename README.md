# WalkRoute

**WalkRoute** is a free walking route planner that works anywhere in the world — no sign-up, no ads, no tracking.

You pick where you want to go, it finds you the best pedestrian path. It knows about sidewalks, crosswalks, footpaths, and parks — not just roads. It also tells you how long the walk takes, how many steps you'll do, and how many calories you'll burn.

It works in English, Spanish, and Catalan.

**Try it here → [https://walking-routes-web.vercel.app/](https://walking-routes-web.vercel.app/)**
