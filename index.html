WalkRoute

A free, privacy-first pedestrian router. No API keys, no billing, no tracking.

  • Real pedestrian paths — sidewalks, footpaths, parks via Valhalla routing
  • Walking metrics — steps and calories, not just distance
  • Multi-stop — up to 5 intermediate stops
  • Alternatives — compare up to 3 walking paths
  • 100% client-side — no server, no accounts
  • Worldwide — works anywhere OpenStreetMap has data
